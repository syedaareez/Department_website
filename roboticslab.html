EIGHTH SEMESTER


IPX-422

CAD/CAM ROBOTICS LABORATORY

[0 0 2 1]

Course Objectives

To expose students for 3D modelling on one of the commercially available software.

To impart training to students on CNC lathe and CNC milling machine.

To train students for operation of robotic arm.


Course Outcomes
  1. Students will be able to make 3D parts for commonly used mechanical components / products, create assembly and finally generate part prints.

  2. Students will be able to create programs for various parts on CNC lathe and milling machine manually and using commercially available software

  3. Students should be able to program robotic arm using teach pendant.

List of Practicals

Creating component drawing and making sub assemblies of components using PRO-Engineer, UNIGRAPHICS /IDEAS/CATIA choosing from the following components.

Steam stop valve

Tail stock

Plummer block

Check valve

Flange Coupling

Flange Coupling

Stuffing Box

Connecting Rod

Programming on CNC lathe and milling, demonstrating linear interpolation, circular interpolation, canned cycles using MASTER CAM/DELCAM etc.

Demonstration of Programming Robotic manipulators
